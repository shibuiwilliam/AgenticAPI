# agenticapi-claude-agent-sdk

> **This extension has been merged into the main `agentharnessapi` package.**
> Install it as an optional extra:
>
> ```bash
> pip install agentharnessapi[claude-agent-sdk]
> ```
>
> The import path is `agenticapi.ext.claude_agent_sdk`:
>
> ```python
> from agenticapi.ext.claude_agent_sdk import ClaudeAgentRunner
> ```

This directory is preserved for historical reference. The source code now
lives at `src/agenticapi/ext/claude_agent_sdk/` and tests at
`tests/unit/ext/claude_agent_sdk/`.

See [example 13](../../examples/13_claude_agent_sdk/) for a working demo.
